## 📌 Current project

### [Steam Games Analytics Pipeline](https://github.com/kiayria/steamscope)
A personal project built to mirror my professional stack on public data.

Pulls data from the Steam API (games, player counts, reviews, pricing history) and processes it through a medallion architecture:

```
Steam API → S3 (bronze) → PySpark + Iceberg (silver) → DuckDB (gold)
```

Orchestrated with Airflow on a daily schedule. Answers questions like:
- Which genres are growing in player count?
- What factors correlate with positive reviews?
- How does sentiment shift over a game's lifetime?
